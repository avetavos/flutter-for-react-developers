In React you reach for CSS or inline styles to add spacing, center content, or arrange items in a row. In Flutter there is no CSS. Every visual concern — padding, centering, clipping, shadows, rows, columns — is expressed as a widget in the tree. This is the single biggest paradigm shift, and once it clicks, the rest of Flutter follows naturally.

## Layout is part of the widget tree

export const reactEverything = `// React — layout via CSS / inline styles
function UserCard({ name, role }) {
  return (
    <div style={{ padding: 16, display: 'flex', flexDirection: 'column', alignItems: 'center' }}>

      <div style={{ color: '#666', marginTop: 4 }}>{role}</div>
    </div>
  );
}`;

export const flutterEverything = `// Flutter — layout via widgets
class UserCard extends StatelessWidget {
  final String name;
  final String role;
  const UserCard({super.key, required this.name, required this.role});

  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: const EdgeInsets.all(16),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          Text(name, style: const TextStyle(fontWeight: FontWeight.bold, fontSize: 20)),
          const SizedBox(height: 4),
          Text(role, style: const TextStyle(color: Colors.grey)),
        ],
      ),
    );
  }
}`;

<Diff title="CSS layout → layout widgets">

- **`Padding` widget** = CSS `padding` property. Wrap any widget: `Padding(padding: EdgeInsets.all(16), child: ...)`.
- **`Center` widget** = CSS `margin: auto` or `align-items: center`. Drop any widget inside it and it centers in available space.
- **`Column` widget** = CSS `display: flex` + `flex-direction: column`. Children stack vertically.
- **`Row` widget** = CSS `display: flex` (default `flex-direction: row`). Children line up horizontally.
- **`SizedBox(height: n)`** = CSS `margin-bottom: n` or `gap: n`. A fixed-size gap between children.
- **No `className` or `style` attribute** — ALL layout lives in the widget tree. The tree is your stylesheet.

</Diff>
