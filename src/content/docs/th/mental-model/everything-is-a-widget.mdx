ใน React เวลาที่คุณต้องการเพิ่ม spacing, จัดตำแหน่ง content หรือเรียง item เป็นแถว คุณจะนึกถึง CSS หรือ inline styles ทันที แต่ใน Flutter ไม่มี CSS ทุกสิ่งที่เกี่ยวกับการแสดงผล — padding, centering, clipping, shadows, rows, columns — ล้วนถูก express เป็น widget ในต้นไม้ นี่คือการเปลี่ยนแปลงทางความคิดที่ใหญ่ที่สุด และเมื่อคุณ "คลิก" กับมันได้ Flutter ที่เหลือก็จะตามมาได้อย่างเป็นธรรมชาติ

## Layout เป็นส่วนหนึ่งของ widget tree

export const reactEverything = `// React — layout via CSS / inline styles
function UserCard({ name, role }) {
  return (
    <div style={{ padding: 16, display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
      <div style={{ fontWeight: 'bold', fontSize: 20 }}>{name}</div>
      <div style={{ color: '#666', marginTop: 4 }}>{role}</div>
    </div>
  );
}`;

export const flutterEverything = `// Flutter — layout via widgets
class UserCard extends StatelessWidget {
  final String name;
  final String role;
  const UserCard({super.key, required this.name, required this.role});

  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: const EdgeInsets.all(16),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          Text(name, style: const TextStyle(fontWeight: FontWeight.bold, fontSize: 20)),
          const SizedBox(height: 4),
          Text(role, style: const TextStyle(color: Colors.grey)),
        ],
      ),
    );
  }
}`;

<Diff title="CSS layout → layout widgets">

- **`Padding` widget** = CSS property `padding` — wrap widget ใดก็ได้: `Padding(padding: EdgeInsets.all(16), child: ...)`
- **`Center` widget** = CSS `margin: auto` หรือ `align-items: center` — วาง widget ใดก็ได้ข้างในแล้วมันจะ center ในพื้นที่ที่มี
- **`Column` widget** = CSS `display: flex` + `flex-direction: column` — children เรียงตามแนวตั้ง
- **`Row` widget** = CSS `display: flex` (default `flex-direction: row`) — children เรียงตามแนวนอน
- **`SizedBox(height: n)`** = CSS `margin-bottom: n` หรือ `gap: n` — ช่องว่างขนาดคงที่ระหว่าง children
- **ไม่มี `className` หรือ `style` attribute** — layout ทั้งหมดอยู่ใน widget tree ต้นไม้คือ stylesheet ของคุณ

</Diff>
