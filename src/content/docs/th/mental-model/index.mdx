Flutter ไม่ใช่ดินแดนแปลกหน้า — มันเหมือนเมืองที่คุ้นเคยแต่ใช้ชื่อถนนต่างออกไป instinct เกี่ยวกับ declarative UI, component composition, props-down/events-up และ state ที่ขับเคลื่อน render ทั้งหมดล้วนใช้ได้ใน Flutter สิ่งที่เปลี่ยนไปคือ syntax layer และ framework convention เพียงไม่กี่อย่าง module นี้จะ map การเปลี่ยนแปลงเหล่านั้นอย่างชัดเจน เพื่อให้คุณไม่ต้องเสียเวลาเรียนพื้นฐานใหม่และเริ่มทำงานได้ทันที

## หกการเปลี่ยนแปลง

| บทเรียน | การเปลี่ยนแปลง |
|---|---|
| **Everything is a Widget** | Layout/spacing/styling เป็น widget ไม่ใช่ CSS |
| **Declarative UI** | แนวคิดเดิม (`UI = f(state)`) — แต่ใช้ reconciler ต่างกัน |
| **No JSX** | Dart constructors แทน angle-bracket tags |
| **Composition** | การ wrap widget แทนการ wrap component |
| **Same vs Different** | ตาราง lookup สำหรับ muscle memory ของคุณ |
